# Concatenum

Concatenum é um projeto pessoal de escrita e curadoria. A ideia é aproximar temas que normalmente aparecem separados — tecnologia, trabalho, cultura, comportamento, notícias — e observar as conexões que surgem quando eles são colocados lado a lado.

Os textos não partem de grandes teses nem buscam conclusões fechadas. Funcionam mais como registros de curiosidade, padrões recorrentes e perguntas que continuam fazendo sentido mesmo quando o contexto muda.

Além da newsletter, o site também abriga coleções paralelas, como registros visuais, anotações de caminhadas e outros temas explorados com a mesma lógica de observação e ligação entre pontos.

## Tema
